# GigShield Dashboard

A React-based insurance dashboard for gig economy workers, featuring risk pool management, fraud detection, claims processing, and policy oversight.

## Tech Stack

- **React 18** — UI framework
- **Vite** — Build tool & dev server
- **Recharts** — Data visualization
- **Lucide React** — Icons

## Features

- **Overview** — Key metrics and summary stats
- **Risk Pool** — Monitor and manage risk pools
- **Claims** — Track and process insurance claims
- **Policy** — Policy management and details
- **Fraud Engine** — Fraud detection and alerts
- **Triggers** — Automated trigger configuration

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
src/
├── components/     # UI components (Claims, FraudEngine, Overview, etc.)
├── data/           # Mock data
├── App.jsx         # Root component with routing/layout
├── main.jsx        # Entry point
└── index.css       # Global styles
```
